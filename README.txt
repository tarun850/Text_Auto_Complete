The program takes 'n' no of words as input and an incomplete word 'w' and prints all the complete words that was initially given as input.
The Program finds and prints all the possible words which can be formed using the incomplete word w.

Input format :
Line 1 : Integer n
Line 2 : n words (separated by space)
Line 3 : Word w

Output Format:
All possible words (in different lines)
